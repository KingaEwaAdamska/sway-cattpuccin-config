# Sway config
Sway configuration used in Fedora

## Install
Make sure you have:
- nm-applet
- blueman-applet
- sway
- wabar
- swaylock
- wofi
- brightnessctl

To use config copy folders to `~/.config ` or clone repo and use:
```
ln -sf <repo_path>/sway ~/.config/sway
ln -sf <repo_path>/waybar ~/.config/waybar
ln -sf <repo_path>/wofi ~/.config/wofi
```
